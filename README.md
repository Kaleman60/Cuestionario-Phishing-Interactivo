# Cuestionario-Phishing-Interactivo

Aplicación web educativa para aprender a identificar correos electrónicos de phishing. El usuario analiza correos reales vs fraudulentos, vota su veredicto y recibe explicaciones detalladas sobre las señales de alerta (errores en el remitente, enlaces sospechosos, diseño fraudulento, etc.).

## Características

- Interfaz intuitiva que simula una bandeja de entrada de correo
- Múltiples escenarios de correos legítimos y de phishing
- Sistema de votación: "¿Es este correo fraude?"
- Explicaciones detalladas de cada señal de alerta
- Seguimiento de puntuación
- Categorías: banco, tiendas online, redes sociales, servicios tecnológicos

## Instalación

```bash
# Clonar el repositorio
git clone <url-repositorio>
cd Cuestionario-Phishing-Interactivo

# Abrir en navegador (sin servidor necesario)
# Simply open index.html in your browser
```

O simplemente abre el archivo `index.html` directamente en tu navegador.

## Cómo usar

1. Selecciona un correo de la bandeja de entrada
2. Analiza el remitente, asunto, contenido y enlaces
3. Vota: "¿Es phishing?" (Sí/No)
4. Consulta la explicación de las señales de alerta
5. Continúa con el siguiente correo
6. Consulta tu puntuación final

## Tecnologías

- HTML5, CSS3, JavaScript vanilla
- Sin dependencias externas

## Señales de alerta que aprenderás a identificar

- Remitente sospechoso o dominio incorrecto
- Errores ortográficos y gramaticales
- Urgencia injustificada
- Enlaces que no coinciden con el texto
- Solicitudes de información personal
- Diseño deficiente o imágenes genéricas
- Saludos genéricos
- Ofertas demasiado buenas para ser ciertas
